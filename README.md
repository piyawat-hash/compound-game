# เกมดอกเบี้ยทบต้น (Compound Interest Game)

โปรเจกต์นี้เป็นเกมเพื่อการเรียนรู้เกี่ยวกับ **ดอกเบี้ยทบต้น**  
พัฒนาด้วยภาษา **HTML, CSS และ JavaScript**  
มีวัตถุประสงค์เพื่อช่วยให้ผู้เรียนเข้าใจหลักการคำนวณดอกเบี้ยทบต้น
ผ่านการเรียนรู้เชิงโต้ตอบ (Interactive Learning)

---

## วัตถุประสงค์ของโครงงาน

1. เพื่อศึกษาหลักการคำนวณดอกเบี้ยทบต้น
2. เพื่อฝึกทักษะการเขียนโปรแกรมเว็บไซต์ด้วย HTML, CSS และ JavaScript
3. เพื่อพัฒนาเกมการศึกษา (Educational Game)
4. เพื่อสร้างสื่อการเรียนรู้ที่เข้าใจง่ายและน่าสนใจ

---

## ขอบเขตของโครงงาน

- ผู้เล่นจะได้รับโจทย์ที่สุ่มเงินต้น อัตราดอกเบี้ย และระยะเวลา
- ผู้เล่นต้องคำนวณและทายจำนวนเงินสุดท้าย

โดยที่  
- P คือ เงินต้น  
- r คือ อัตราดอกเบี้ยต่อปี  
- t คือ จำนวนปี  
- A คือ จำนวนเงินรวมสุดท้าย  

---

## คุณสมบัติของโปรแกรม

- เกมแบบเป็นด่าน (Level)
- ระบบคะแนน (Score)
- สุ่มโจทย์อัตโนมัติ
- ตรวจสอบคำตอบและแสดงผลลัพธ์
- แสดงตารางเงินเพิ่มรายปี
- รองรับการใช้งานบนโทรศัพท์มือถือ
- ใช้งานผ่านเว็บเบราว์เซอร์ได้โดยไม่ต้องติดตั้งโปรแกรม

---

## เครื่องมือและเทคโนโลยีที่ใช้

- HTML : โครงสร้างเว็บไซต์
- CSS : ตกแต่งรูปแบบเว็บไซต์
- JavaScript : ควบคุมการทำงานของเกม
- GitHub Pages : ใช้เผยแพร่เว็บไซต์

---

## วิธีการใช้งานโปรแกรม

1. เปิดเว็บไซต์เกมดอกเบี้ยทบต้น
2. ระบบจะแสดงเงินต้น อัตราดอกเบี้ย และจำนวนปี
3. ผู้เล่นกรอกจำนวนเงินสุดท้ายที่คาดว่าจะได้รับ
4. กดปุ่มส่งคำตอบ
5. ระบบจะแสดงเฉลย คะแนน และตารางรายปี
6. เข้าสู่ด่านถัดไปโดยอัตโนมัติ

---

## ประโยชน์ที่ได้รับจากโครงงาน

- เข้าใจการคำนวณดอกเบี้ยทบต้นได้ชัดเจนยิ่งขึ้น
- ฝึกการคิดวิเคราะห์และการประมาณค่า
- ฝึกทักษะการพัฒนาเว็บไซต์
- สามารถนำความรู้ไปประยุกต์ใช้ในชีวิตจริงด้านการเงิน
